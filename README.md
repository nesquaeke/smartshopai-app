# SmartShopAI

Production-ready Next.js App Router foundation for a community deal-sharing and price-tracking experience.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Zustand
- TanStack Query
- Recharts

## Run locally

1. Install Node.js 20+.
2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Implemented in this phase

- Glassmorphism design tokens, mesh background, dark mode-first foundation
- Sticky premium navbar with search, categories panel, notifications, profile, and CTA
- Multi-level scalable category menu with trending and recently visited sections
- Interactive product cards with mock upvote/downvote and animated score
- Dynamic rank-based home feed (bento-style prominence)
- Typed mock datasets for categories, products, and price history
